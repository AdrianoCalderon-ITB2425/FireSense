Per veure les dades del teu node IoT al mapa, segueix aquests passos per vincular el teu dispositiu RAK4631 al teu compte FireSense.

1
Desplegament físic del node
Col·loca el node RAK4631 a la zona forestal a monitoritzar. Assegura't que té visibilitat al gateway LoRaWAN més proper (radi màxim 15km).
2
Obté el Device EUI del teu node
El Device EUI és l'identificador únic del teu dispositiu. El trobaràs imprès a l'etiqueta del RAK4631 o a través del firmware serial.
3
Vincula el node al teu compte
Introdueix el Device EUI aquí baix. Un cop verificat, apareixerà al mapa en temps real amb les seves dades de temperatura i humitat.
