// PAS 1 — INSTAL·LA CHIRPSTACK (servidor propi)
1
Instal·la ChirpStack amb DockerAl teu servidor Linux: curl -fsSL https://get.docker.com | sh
Consulta el manual PDF per als fitxers de configuració complets (docker-compose.yml, chirpstack.toml, eu868.toml).
2
Obté la IP del teu servidorip addr show | grep "inet " | grep -v 127.0.0.1
Anota aquesta IP — la necessitaràs al gateway i Node-RED.
3
Crea Device Profile + Application a ChirpStackDevice Profiles → Add → Regió: EU868 · MAC: LoRaWAN 1.0.3 · OTAA · Interval: 3600s · Codec 7 bytes (del manual)
Applications → Add → firesense-nodes
4
Registra el node i genera la AppKeyAdd device → DevEUI de l'etiqueta (sense dos punts)
OTAA Keys → genera AppKey → copia-la (la necessites al firmware)
5
Verifica que ChirpStack funcionaAccedeix a http://TU_IP:8080 i comprova que veus el panel sense errors.
// CHIRPSTACK CLOUD FIRESENSE
1
Accedeix al panel FireSense CloudCredencials rebudes per email:
https://chirpstack.firesense.io
2
Registra el teu nodeApplications → firesense-nodes → Add device
DevEUI (sense dos punts) · Genera AppKey · copia-la per al firmware
3
Configura el gatewayMQTT Broker: mqtt.firesense.io · Port: 1883
// PAS 2 — PROGRAMA EL FIRMWARE I CONFIGURA EL GATEWAY
1
Programa el firmware al RAK4631Arduino IDE → codi del manual PDF · Substitueix nodeDevEui i nodeAppKey (generada a ChirpStack)
Conversió DevEUI: AC:1F:09:FF → {0xAC, 0x1F, 0x09, 0xFF...}
2
Upload i verifica al Monitor Serie (115200 baud)Veuràs JoinRequest. El JoinAccept arribarà quan el gateway estigui Online — és normal no veure'l encara.
3
Configura el gateway RAK7289V2Accedeix a http://192.168.1.1 (root/root)
LoRa → Network Settings → Protocol: LoRa Gateway MQTT Bridge · ChirpStack 4.x
MQTT Broker: IP del teu servidor ChirpStack · Port: 1883
4
Configura Node-REDImporta el JSON del manual PDF · Doble clic "MQTT ChirpStack" → canvia localhost per la IP del servidor
Doble clic "InfluxDB" → URL, token i bucket sensor_data
5
Verifica que tot funciona ✓ChirpStack → Gateways: punt verd ✓
ChirpStack → Device → LoRaWAN Frames: JoinRequest + JoinAccept ✓
InfluxDB bucket sensor_data conté dades ✓
